About Fetcher Testing

Test LLM-generated regex patterns for parsing transaction emails. This tool uses Gemini AI to analyze your email text and automatically generate patterns to extract transaction amounts, merchants, and currencies.

How it works:

  • Configure your fetcher with name, from emails, and default currency
  • Add one or more email examples (raw text or Gmail message IDs)
  • For providers with multiple email formats (e.g., Wise), add examples of each format
  • Gemini AI generates regex patterns that work across all provided examples
  • Test patterns against real emails from Gmail
  • Save the fetcher configuration to database for future use
