Protect Your Data

Register a passkey to enable end-to-end encryption. Your transaction data will be encrypted so only you can read it.

This step is optional — you can set up encryption later from Passkey Setup.

Register Passkey

How it works: A passkey on your device derives an encryption key using the PRF extension. The server uses this key to encrypt your data at rest. You'll authenticate with your passkey each session to provide the key.

Tips

  • Works best with built-in authenticators (Touch ID, Windows Hello, Android biometrics)
  • You can add more passkeys later from the Encryption Setup page
  • If you skip, your data is still stored securely — just not end-to-end encrypted
