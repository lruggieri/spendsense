What does this do?

This feature fetches transaction data from your Gmail account by scanning emails from configured sources. The data is automatically parsed and saved to persistent storage for classification.

Note: Your Gmail credentials from login will be used automatically - no additional authorization needed!

How it works:

  1. Select which sources to fetch (or leave all checked for all sources)
  2. Choose the start date for fetching emails
  3. Click "Start Fetching Transactions"
  4. The app will scan your Gmail using your existing credentials
  5. Transactions will be automatically parsed and saved
  6. Progress will be displayed in real-time
