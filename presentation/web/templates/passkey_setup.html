Set Up End-to-End Encryption

Register a passkey to prepare for end-to-end encryption. Once encryption is fully enabled, only you will be able to read your financial data — not even the server.

How it works: A passkey on your device derives an encryption key using the PRF extension. The server uses this key to encrypt your data at rest — without it, stored data cannot be read. You'll need to authenticate with your passkey each session to provide the key.
